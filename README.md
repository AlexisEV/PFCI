# Implementación de Artículos - PFC I

Este repositorio contiene las implementaciones de los artículos seleccionados para el curso **Proyecto Final de Carrera I (PFC I)**.  
El objetivo es reproducir y comparar modelos de Deep Learning aplicados a la **detección y clasificación de lesiones cutáneas**.

## Artículos Implementados

### 1. Chatterjee et al. (2024)
**Título:** *Early Detection of Multiclass Skin Lesions Using Transfer Learning-Based IncepX-Ensemble Model*  
**Descripción:**  
Propone un modelo Ensemble entre **InceptionV3** y **Xception** denominado **IncepX-Ensemble**, que utiliza **transfer learning** y técnicas de **data augmentation** para el conjunto de datos **HAM10000**.  

---

### 2. Hao et al. (2023)
**Título:** *ConvNeXt-ST-AFF: A Novel Skin Disease Classification Model Based on Fusion of ConvNeXt and Swin Transformer*  
**Descripción:**  
Presenta un modelo híbrido basado en la **fusión de redes convolucionales (ConvNeXt)** y **Transformers (Swin Transformer)** mediante los módulos de atención **AFF (Attentional Feature Fusion)** y **ECA (Efficient Channel Attention)**.
